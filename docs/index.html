Airline Sentiment Analysis

Deep learning classification of US airline tweets into positive, neutral, and negative

View on GitHub View Report

Overview

This project explores customer sentiment in airline tweets using a neural network built with TensorFlow/Keras. It includes preprocessing, modeling, evaluation, and visualizations of performance metrics.

  • Data sources: Tweets CSV and SQLite mirror
  • Targets: positive, neutral, negative
  • Outputs: accuracy/loss plots and an interactive HTML report

How to reproduce

  1. Clone the repo and install deps: pip install -r requirements.txt
  2. Open Sentiment Analysis.ipynb and run all cells
  3. Artifacts will include plots and Sentiment Analysis.html
